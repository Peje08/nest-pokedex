# Ejecutar en Desarrollo

1. Clonar el repositorio
2. Ejecutar:
```
   npm install
```
Si usas yarn:
```
    yarn install 
```
3. Tener Nest CLI instalado
```
    npm i -g @nest/cli
```
4. Levanatr la BD
```
  docker-compose up -d
```

## Stack usado
* MongoDB
* Nest
